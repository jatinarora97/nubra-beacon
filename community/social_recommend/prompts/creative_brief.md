# Creative Brief Prompt

Create a designer-ready brief for a post/carousel/reel.

Include:

- Audience
- User pain
- Core message
- Visual direction
- Frame-by-frame structure if carousel/video
- Required product context
- What to avoid
- Evidence snippets

Design rules:

- One idea per frame.
- Avoid crowded trading screens.
- Show risk context where trading execution is involved.
- Do not imply profit, prediction or trade recommendation.
