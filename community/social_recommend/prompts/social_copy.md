# Social Copy Prompt

Write concise social copy for the selected recommendation.

Tone:

- clear
- product-focused
- educational
- practical
- not salesy

Constraints:

- No investment advice.
- No profit guarantees.
- No unsupported competitor claims.
- Only claim Nubra capabilities that appear in the active Nubra context.
- Upcoming features must be phrased as upcoming/planned.

Structure:

1. Hook based on user pain
2. Brief explanation
3. Practical checklist or workflow
4. Nubra context if claim-safe
5. Soft CTA
